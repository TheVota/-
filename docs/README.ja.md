# VSPO Fan Schedule API

VSPO Fan Schedule API は、公開されている VSPO! 関連の配信情報を扱うための非公式 Rails API サーバーです。

このプロジェクトは [YunzheZJU/non-stop-story](https://github.com/YunzheZJU/non-stop-story) を元にしています。元プロジェクトは MIT License で公開されています。著作権表示とライセンス文は `LICENSE` を確認してください。

このリポジトリは VSPO!、Virtual Entertainment、Brave group、YouTube、VTuber Live の公式プロジェクトではありません。各名称・商標・サービス名はそれぞれの権利者に帰属します。

公開する前に、`config/master.key`、`.env`、実際のAPIキー、SMTPパスワード、本番用credentials、配布権限のないアイコンや画像が含まれていないことを確認してください。
